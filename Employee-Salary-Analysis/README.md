# 💰 Employee Salary Analysis

## 📌 Overview
This project analyzes employee salary data to understand patterns and trends.

## 🧠 Approach
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of salary distribution  

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Insights
- Salary distribution across different roles  
- Factors affecting employee salaries  

## 🚀 Key Learning
- Data visualization  
- Exploratory data analysis
