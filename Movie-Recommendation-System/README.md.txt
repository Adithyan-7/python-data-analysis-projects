# 🎬 Movie Recommendation System

## 📌 Overview
This project recommends movies based on similarity using Machine Learning techniques.

## 🧠 Approach
- Data cleaning using Pandas
- Feature extraction using CountVectorizer
- Similarity calculation using cosine similarity

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 🚀 How it works
Enter a movie name → system recommends similar movies

## 📊 Example
Input: Dune  
Output: Interstellar, The Matrix, etc.
