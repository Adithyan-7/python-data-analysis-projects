# Python Data Analysis Projects

A collection of end-to-end data analysis and machine learning projects built with Python and Jupyter Notebook. Each project covers real-world datasets with full EDA, preprocessing, modelling, and insights.

## Projects

| Project | Description | Techniques Used |
|---------|-------------|-----------------|
| 🚗 Car Price Prediction | Predicts used car prices based on features like brand, age, mileage | Regression, Feature Engineering |
| 🏠 House Price Prediction | Estimates property values using structural and location features | Linear Regression, EDA |
| 👔 Employee Salary Analysis | Analyses salary distribution across roles, departments, and experience | EDA, Visualization |
| 🔁 Employee Retention Analysis | Identifies factors that influence employee turnover | Classification, Correlation Analysis |
| 🎬 Movie Recommendation System | Recommends movies based on user preferences | Content-Based Filtering |
| 🛒 Purchase Recommendation | Suggests products based on buying patterns | Collaborative Filtering |
| 🌸 Iris Flower Classification | Classic ML classification on the Iris dataset | KNN, Decision Tree, SVM |
| 🧾 Expense Tracker | Analyses personal spending patterns and trends | Pandas, Matplotlib |
| 🍽️ Recipe Analysis | Explores nutritional content and recipe patterns | EDA, Seaborn |
| 🚖 Uber Ride Analysis | Analyses ride data for trends in pickup, time, and distance | Time Series, Visualization |
| 📚 Student Marks Analysis | Examines academic performance patterns | Statistical Analysis |

## Tech Stack

- **Language:** Python 3
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Environment:** Jupyter Notebook

## How to Run

```bash
git clone https://github.com/Adithyan-7/python-data-analysis-projects.git
cd python-data-analysis-projects
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
jupyter notebook
```

Then open any `.ipynb` file from the project folder of your choice.
