# 🚗 Car Price Prediction

## 📌 Overview
This project predicts car prices based on features such as present price, fuel type, selling type, and transmission.

## 🧠 Approach
- Data preprocessing and cleaning using Pandas  
- Feature selection and encoding  
- Model training using Linear Regression  

## ⚙️ Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Results
Built a regression model to predict car prices with good accuracy.

## 🚀 Key Learning
- Regression modeling  
- Feature importance  
- Data preprocessing techniques
