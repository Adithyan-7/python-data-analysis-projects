# 🏠 House Price Prediction

## 📌 Overview
This project predicts house prices based on various features such as area, location, and number of rooms.

## 🧠 Approach
- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Model building using regression techniques  

## ⚙️ Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Results
Developed a model to estimate house prices based on input features.

## 🚀 Key Learning
- Regression models  
- Feature engineering  
- Data preprocessing
