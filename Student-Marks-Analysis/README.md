# 📊 Student Marks Analysis

## 📌 Overview
This project analyzes student marks data to identify performance trends and insights.

## 🧠 Approach
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of student performance  

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Insights
- Subject-wise performance  
- Overall student trends  

## 🚀 Key Learning
- Data analysis  
- Visualization techniques
